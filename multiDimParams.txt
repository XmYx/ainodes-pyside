You can make use of all those parameters in a prompt
any of the params will accept lists of values like
--steps=10,20,30,40
if you use more than one param they get rendered in any possible combination so be aware the number of images grows fast if you add many params with many values
For now see this as a very advanced feature as you would need to know valid values for the params, over the curdse of the next few days we will add some more information here
